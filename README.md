# Friend-Finder

A compatibility-based app that will compare users answers to determine their best friend match. 
![image](https://github.com/shayshae5482/Responsive-Portfolio/blob/master/assets/images/friendfinderscreen.png)
 

# Technologies Used:
* JavaScript
* HTML
* CSS
* Bootstrap
* Materialize
* Express
* Body-parser
* Node.js
